# elite-tool

## WTF is this

A simple, dirty and quick tool for a friend's repairment shop.

## Usage

Run the server with:

```bash
git clone https://github.com/gorcajo/elite-tool.git
cd elite-tool
sudo ./run.sh
```

And then you can browse to <http://localhost/pages/new>
